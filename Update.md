## Updating Bash scripts to support Joomla in DDEV

There will be regular updates for the scripts that were installed during installation. To update these scripts you need to do the following.

Enter the following command in a terminal:

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/renekreijveld/ddev-joomla/refs/heads/main/updater/ddev-joomla-updater.sh)"
```

This will update the installed scripts.
